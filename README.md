# 📱 Chuỗi Xanh Việt

Mobile application built with **Expo (React Native)** to support green lifestyle initiatives 🌱.

---

## 🚀 Getting Started

### 1. Clone repository

```bash
git clone https://github.com/MeowMeow-org/chuoi-xanh-viet-mobile.git
cd chuoi-xanh-viet-mobile
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the app

```bash
npx expo start
```

---

## 📱 Run on Device

- 📱 Scan QR code with **Expo Go (Android/iOS)**
- 💻 Or press:
  - `a` → Run on Android Emulator
  - `w` → Run on Web

---

## ⚙️ Tech Stack

- ⚛️ Expo (React Native)
- 🧭 Expo Router
- 🟦 TypeScript
- 🎨 NativeWind (Tailwind CSS for React Native)
- 🧩 gluestack-ui (UI components)
